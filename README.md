# AI Workspace

AI Workspace is a full-stack Generative AI project built with **Node.js**, **TypeScript**, **React**, **Vite**, and **OpenAI APIs**.  
It supports **RAG (Retrieval-Augmented Generation)**, document uploads, embeddings, memory-based chat, and a clean frontend UI.
